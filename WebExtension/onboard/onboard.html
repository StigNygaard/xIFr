xIFr version has been installed.

With this tool you can see a lot of the metadata that can be embedded in JPEG files. Various data of types EXIF, IPTC and XMP are supported.

To see (if there are) any recognized metadata in an image, you can usually just right-click the image and choose "View EXIF data" in your browser's contextmenu. But to learn xIFr better to know, and especially to see what the "Deep Search" feature is all about, I have also created a little introduction or demonstration page:

www.rockland.dk/xIFr/start

Also, make sure to make a visit to xIFr's Options Page where you can adjust some aspects of how xIFr looks and behaves. Look for the wheel in the metadata popup to access the options. My personal favorites are to use "Dark Theme" and "Snap right" popup position.

New in xIFr version 2.3: When an image is available in multiple image-formats, xIFr will look for jpeg-version where it can parse meta-data from, even if another format has been chosen by browser to show on webpage. See www.rockland.dk/xIFr/start for an example.

I hope you'll find xIFr useful and enjoy using it...
